Truenas Server

What is TrueNAS?

TrueNAS is a powerful, open-source storage operating system that provides enterprise-grade storage capabilities. It is designed to be used in both home and business environments, offering features such as data protection, scalability, and high availability. TrueNAS supports various storage protocols, including SMB, NFS, iSCSI, and more, making it a versatile solution for different storage needs.

What do I use TrueNAS for?

I use TrueNAS as a central storage solution for my home network. It serves as a file and media server, allowing me to store and access files from multiple devices on my network. TrueNAS also provides data protection features such as snapshots and replication, ensuring that my data is safe and secure. I have 3 HDDS in Raid 5 as my main file storage. Raid 5 means I have 2 drives joined together while the 3rd drive acts as a fall back drive in case a drive failure occurs. I then use 5 800gb SAS SSD drives in Raid 0 for video editing my personal youtube channel.

What am I hosting

My TrueNAS server hosts a couple of apps through the truenas app setting, and then through portainer.io I have a few docker containers running. Currently I have ClamAV (antivirus), metube(YouTube clone), Portainer.IO (docker container) and a few other apps running. In Portainer I have currently running Home Assistant (Self Hosted Home Automation and analytics).
